# Yanyn Email
The official client of Yanyn Email is based on the Qt6 and supports QQ Email, 163 Email, Outlook Email, Gmail Email, etc. 
 


  
## v26.1 Version a0226 Patch Update
 
 
- The email sending and receiving function was implemented using Mailio.
- The thread issue was optimized. 
 
 
## Technology
 
 
- C++
- Qt 6
- Mailio
- Boost
- OpenSSL
