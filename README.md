# Yanyn Email
晏阳邮箱的官方客户端，基于Qt6框架，支持QQ邮箱、163邮箱、Outlook邮箱、Gmail邮箱等。
 
 


  
## v26.1版本 a0226补丁 更新内容
 
 
- 使用Mailio实现了邮件收发功能
- 优化了线程问题
 
 
## 技术栈
 
 
- C++
- Qt 6
- Mailio
- Boost
- OpenSSL
